# Canonical Running Example

Alice was CEO of Acme Corp from 2020-01-01 to 2023-03-31. The source is Acme 2020 Annual Report. Bob verified the statement on 2023-04-05. Confidence is 0.93. Alice was appointed by the Board of Directors, and the appointment happened in New York.

This is not a simple binary edge. Claim context includes role, employer, start and end dates, appointment authority, and location. Provenance includes source document, verifier, and verification date. Confidence is a quality score about the claim.
