# Comparison Matrix

| Model | Best fit | Awkward fit | Provenance |
| --- | --- | --- | --- |
| RDF | semantic interoperability | high-speed operational traversal | strong with n-ary or named graphs |
| LPG | traversal and analytics | global semantics | good with event nodes |
| RDF-star / RDF 1.2 | compact statement metadata | uneven parser support | strong |
| Hyper-relational | ranked claims and references | custom app semantics | strong |
| Hypergraph | n-ary relations | binary graph tooling | explicit participants |
| Datalog | recursive derived facts | physical storage | lineage can be tracked |
| Topic Maps | subject identity | modern ecosystem size | occurrence based |
| EAV | sparse attributes | validation and joins | weak unless modeled carefully |
| Hybrid graph + vector | source-aware retrieval | truth from similarity alone | graph citations retained |
