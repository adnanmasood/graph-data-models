# RDF-star / RDF 1.2 Examples

RDF-star and RDF 1.2 annotation syntax aim to make statement-level metadata more compact than classic reification. Tool support varies, so the executable code keeps syntax files as inspectable examples and tests a fallback lowering into the RDF n-ary pattern.

Run it with `python -m graph_model_examples run rdf-star`.

Elegant: source, verifier, and confidence can be written near the statement. Awkward: parser support differs across libraries and standards versions. Conversion loss is controlled here by lowering into an explicit employment role node.
