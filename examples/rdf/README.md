# RDF Examples

This example models the Alice/Acme CEO claim four ways: plain triples, an n-ary employment role node, named graphs, and classic RDF reification.

Run it with `make demo-rdf`.

Elegant: RDF gives global identifiers, standard syntax, and SPARQL queryability. Awkward: binary triples need extra modeling patterns for source, dates, verifier, and confidence. Gotchas: reification is verbose, and a lossy binary edge cannot preserve all claim context during conversion.
